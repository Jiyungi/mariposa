# CPT Codes — Fertility Evaluation & Treatment

Real CPT/diagnosis codes used in a fertility workup. The Grok Voice agent asks the clinic/
insurance about these by code, and the doctor summary can list them. Codes are public coding
references (not PHI).

## Diagnostic test codes

| CPT | Procedure | Coded for |
| --- | --- | --- |
| 89320 | Semen analysis; volume, count, motility, and differential | Male |
| 89322 | Semen analysis; with strict morphology | Male |
| 76830 | Ultrasound, transvaginal | Female |
| 76856 | Ultrasound, pelvic (non-obstetric), complete | Female |
| 58340 | Catheterization/saline infusion for HSG (sono) | Female |
| 74740 | Hysterosalpingography (HSG), radiologic exam | Female |
| 84702 | hCG, quantitative (pregnancy/serum) | Female |

> Coding rule of thumb (real, from ASRM): procedures on the oocyte/embryo are coded for the
> female partner; procedures on sperm alone are coded for the male partner.
